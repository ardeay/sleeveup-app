What is Sleeveup?
A DECK BUILDING AAP FOR MAGIC THE GATHERING!
It's open source, so it's going to get better and better! Sleeveup leverages EchoMTG's amazing API for deck building.

Sleeveup.org was started by @iammenasco. He was searching to contribute to an open source project, and ran into EchoMTG's API. He reached out to @teeg, talked for a few minutes, and @iammenasco started hacking away at this amazing code base with @teeg!

There are many ways to contribute! Join us in making the best MTG resource out there!
